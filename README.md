<h2 id="overview">📌 Overview</h2>

<p>
This project analyzes Swiggy restaurant and order data using 
Excel, Power BI, Python, and Jupyter Notebook. 
The project focuses on restaurant performance, order trends,
ratings analysis, pricing insights, and customer ordering behavior.
</p>

<hr>

<h2 id="objectives">📊 Project Objectives</h2>

<ul>
  <li>Analyze restaurant performance across cities</li>
  <li>Track monthly order trends</li>
  <li>Measure rating distribution</li>
  <li>Visualize price range analysis</li>
  <li>Create interactive business dashboards</li>
</ul>

<h2 id="insights">📌 Key Insights</h2>

<ul>
  <li>Bengaluru has the highest restaurant count</li>
  <li>Most orders are between ₹200–₹500</li>
  <li>McDonald's and KFC received highest orders</li>
  <li>Recommended category dominates restaurant share</li>
  <li>Ratings mostly fall above 4.0</li>
</ul>
